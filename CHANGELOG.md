## [1.0.0] - 2025-08-10
### added
- minimal functional version of tic-tac-toe:
  - 3x3 board
  - two players (x and o)
  - move validation
  - detection of win, draw, or in-progress state
- move input via `row,column` or `row column`
- basic status and end-game messages
